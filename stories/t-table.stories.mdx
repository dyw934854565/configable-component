# 使用配置完成表格组件的调用

## 原来的写法

<Preview>
  <Story name="原来的写法">
  {
    {
      template: `
        <el-table
          :data="tableData">
          <el-table-column
            prop="date"
            label="日期"
            width="180">
          </el-table-column>
          <el-table-column
            prop="name"
            label="姓名"
            width="180">
          </el-table-column>
        </el-table>
      `,
      data() {
        return {
          tableData: [{
            date: '2016-05-02 19:00:00',
            name: '王小虎'
          }, {
            date: '2016-05-03 19:00:00',
            name: '王小虎'
          }]
        }
      }
    }
  }
  </Story>
</Preview>

痛点：手写模版；如果用v-for，有些特殊的列可能不支持；如果有数据是需要改动的就需要手写template

## 改版之后用法

<Preview>
  <Story name="改版之后写法">
  {
    {
      template: `
        <t-table :cols="cols" :data="tableData" style="width: 100%">
        </t-table>
      `,
      data() {
        return {
          cols: [{
            prop: 'date',
            label: '日期',
            extra: {
              width: '180'
            }
          }, {
            prop: 'name',
            label: '姓名',
            extra: {
              width: '180'
            }
          }],
          tableData: [{
            date: '2016-05-02 19:00:00',
            name: '王小虎'
          }, {
            date: '2016-05-03 19:00:00',
            name: '王小虎'
          }]
        }
      }
    }
  }
  </Story>
</Preview>

优点：配置型使用，更方便；el-table的属性都能支持；Table-column的配置可以放在extra中；值需要变化时可以用value方法即可

## 使用组件

表格中经常用到链接和按钮，准备了内置组件，快速完成

<Preview>
  <Story name="内置组件">
  {
    {
      template: `
        <t-table :cols="cols" :data="tableData" style="width: 100%">
        </t-table>
      `,
      data() {
        return {
          cols: [{
            prop: 'date',
            label: '日期',
            extra: {
              width: '180'
            }
          }, {
            prop: 'name',
            label: '姓名',
            extra: {
              width: '180'
            }
          }, {
            prop: 't-btns',
            label: 't-btns',
            component: 't-btns',
            value: (val, scope) => {
              return {
                btns: [
                  {
                    label: '复制',
                    handle: async scope => {
                      // 按钮被点击的处理方法
                      await this.$confirm('确认复制', '提醒')
                    },
                    extra: {
                      // 其余参数，参考el-button
                      size: 'mini'
                    }
                  },
                  {
                    label: '删除',
                    handle: async scope => {
                      // 按钮被点击的处理方法
                      await this.$confirm('确认删除', '提醒')
                    },
                    extra: {
                      // 其余参数，参考el-button
                      size: 'mini',
                      type: 'danger'
                    }
                  }
                ],
                scope
              }
            },
            extra: {
              width: '200'
            }
          }, {
            prop: 't-switch',
            label: 't-switch',
            component: 't-switch',
            value: (val, scope) => {
              return {
                value: Boolean(val),
                handle: async (val, scope) => {
                  // 按钮被点击的处理方法
                  console.log(val)
                  await this.$alert('switch change: ' + val)
                  // 实际场景是更新远程数据再改本地数据或重新获取数据刷新
                  this.tableData[scope.$index][scope.column.property] = Number(val)
                },
                scope
              }
            }
          }, {
            prop: 't-link',
            label: 't-link',
            component: 't-link',
            value: val => {
              return {
                label: '查看详情',
                icon: 'el-icon-view',
                href: 'javascript: void 0;'
              }
            },
            extra: {
              width: '180'
            }
          }, {
            prop: 't-tag',
            label: 't-tag',
            component: 't-tag',
            value: val => {
              const typeMap = {'家': 'success', '公司': 'info'}
              return {
                label: val,
                type: typeMap[val]
              }
            }
          }],
          tableData: [{
            date: '2016-05-02 19:00:00',
            name: '王小虎',
            't-switch': 1,
            't-tag': '家'
          }, {
            date: '2016-05-03 19:00:00',
            name: '王小虎',
            't-switch': 0,
            't-tag': '公司'
          }]
        }
      }
    }
  }
  </Story>
</Preview>


# 自定义组件
当组件不能完全满足需求的时候可以，自己开发一个组件。可能组件需要稍微改动一下，事件监听，变成传callback更方便一些

写完可以和表单组件一样， 直接improt然后传给component。如果使用频繁，可以用Vue.component挂在全局组件上。
